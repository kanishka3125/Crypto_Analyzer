# CryptoAnalyzer - Bitcoin Price Forecasting

A machine learning application for Bitcoin price prediction using Facebook's Prophet model and Flask web dashboard.

## üöÄ Installation

### Step 1: Clone Repository
```bash
git clone https://github.com/kanishka3125/Crypto_Analyzer.git
cd Crypto_Analyzer
```

### Step 2: Create Virtual Environment
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4: Run Application
```bash
python app.py
```

### Step 5: Access Dashboard
Open browser and go to: `http://localhost:5000`
